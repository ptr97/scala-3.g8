# scala-3.g8

A [Giter8][g8] template for a minimal [Scala 3] project.

Template usage
--------------
Using `sbt`, do:

```
sbt new file://../scala-3.g8/
```

in the folder where you want to clone the template.

Using `g8`, do:

```
g8 file://../scala-3.g8/
```

[g8]: http://www.foundweekends.org/giter8/

[Scala 3]: http://dotty.epfl.ch/
